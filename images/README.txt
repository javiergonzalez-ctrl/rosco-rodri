CARPETA DE IMÁGENES — ROSCO DESPEDIDA

Cómo funciona:
- Mete aquí una foto por letra. Cuando se acierte esa letra, la foto sale a pantalla con confetti.
- Nombra el archivo con la letra en MAYÚSCULA y la extensión que tenga: jpg, jpeg, png, webp, gif.
- Para la Ñ usa el nombre N_TILDE (ej: N_TILDE.jpg) o Ñ.jpg si tu sistema lo permite.

Ejemplos:
  A.jpg     → foto de Andrea Abad
  B.png     → foto de Blanca
  D.jpg     → cartel de Torrente / Dani Güiza
  L.webp    → foto de Luis Aragonés
  N_TILDE.jpg → meme de Javi González
  Z.jpg     → la que tú elijas

Si una letra no tiene foto, no pasa nada — el juego sigue normal.

Después de añadir/cambiar fotos, vuelve a desplegar:
  surge .

(o el comando que te genere el deploy)
